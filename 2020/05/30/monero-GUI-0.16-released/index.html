Monero GUI 更新版本 v0.16.0.0（代號：氮星雲）

原文作者：ErCiccione, selsta

更新總覽

這是 Monero GUI 錢包軟體的 v0.16.0.0 主版本發布。

最新的 CLI 錢包軟體發布可以在先前的文章找到。

關於此項版本的數個重點項目:

  • 簡易模式：改善節點尋找演算法
  • UX：顯示預估轉帳手續費
  • UX：新增更新下載與驗證功能
  • UX：新增錢包快取檔自動儲存功能
  • UI：重新設計轉帳功能頁面的進階選項
  • UI：改善節點同步進度條顯示
  • UI：新的語言選擇欄呈現
  • UI：新的載入動畫
  • UI：重新設計設定頁面
  • Trezor：支援新的密碼輸入機制
  • 設定精靈：新增支援種子碼偏移
  • 許多錯誤修復與效能改善

完整的程式碼修改紀錄在可至GitHub查看。
