Monero 發布新版本 v0.18.3.1（代號：氟費米）

原文作者：selsta
更新總覽
====

這是 Monero 軟體的 v0.18.3.1 小數點版本發布，此版本改善了錢包重新整理機制並包含了一些重要的錯誤修復。

關於此項版本的數個重點項目:

  • 優化錢包重新整理機制，減少持續的 RPC 呼叫（#8800）
  • 節點：處理命令列標記在配置文件中不允許的情況（#8766）
  • 節點：確保基本費用不為 0（#8851）
  • 節點：通過正確終止中斷的 TCP 連接來修復在某些系統上的高 CPU 使用率（#8900）
  • 節點：丟棄發送重複交易的連接節點（#8916）
  • 節點：修復長期塊有關權重緩存中的錯誤（#9014）
  • 錢包：改善 scan_tx 重新掃描（#8566）
  • 錢包：不要使用 DNS 來確定是否為本地地址（#8878）
  • 錢包：修復多重簽名錢包中的凍結功能（#8953）
  • 錢包：錢包保存和密碼更改相關的程式碼改進（#8938, #8999）
  • RPC：回傳已提交塊的 ID（#8891）
  • RPC：允許從多重簽名種子恢復錢包（#8942）
  • RPC：添加 –no-initial-sync 標誌，在刷新時分塊刷新以保持可回應狀態（#8941）
  • 在 x86 上加快 perf_timer 初始化，將 Monero 啟動延遲減少 1 秒（#8895）
  • 修復從頭開始挖礦時 Random 的初始化（#8831）
  • 修復 32 位元系統上的金額溢位檢測（#8844）
  • 更新 openssl 至 1.1.1u（#8883）
  • 向 ZMQ 添加 CLSAG 序列化（#8908）
  • 即使禁用服務器驗證，也要設置 SSL SNI（#8899）
  • 修復了一些次要的錯誤並進行了改進

完整的程式碼修改紀錄在可至GitHub查看。
